# Bharat AI - Backend for Vercel

This is the backend/API server for Bharat AI mobile app.

## Deploy to Vercel

1. Push this folder to GitHub
2. Import to Vercel
3. Add environment variables:
   - GOOGLE_CLIENT_ID
   - GOOGLE_CLIENT_SECRET
   - NEXTAUTH_URL (your Vercel URL)
   - NEXTAUTH_SECRET
   - MONGODB_URI
   - CLOUDINARY_CLOUD_NAME
   - CLOUDINARY_API_KEY
   - CLOUDINARY_API_SECRET
   - ADMIN_USERNAME
   - ADMIN_PASSWORD
   - VAPID_PUBLIC_KEY
   - VAPID_PRIVATE_KEY

4. Deploy!

After deployment, update the mobile app with your Vercel URL.
